## 📌 Overview
이 프로젝트는 **안재희**님만을 위해 개발된 1:1 전용 서프라이즈 패키지입니다. 
팀 내 최고의 분위기 메이커이자 재능 넘치는 재희님께 감사의 마음을 담아 릴리즈되었습니다.

## 🎁 Feature List
- **Acrostic-Poem-Module**: 재희님 이름으로 만든 특별한 삼행시 엔진 탑재
- **Compliment-Card-System**: 재희님의 장점들을 분석한 데이터 시각화
- **Interactive-Gift-Game**: 마니또의 정체를 숨긴 미니 클릭 게임

## 🛠 Installation
본 패키지는 재희님의 미소 하나면 즉시 설치가 완료됩니다.
